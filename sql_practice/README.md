# sql_practice

SQL exercises and R-to-SQL translations using DuckDB and a local LCS dataset.

## Files

- `01_window_functions.sql` / `.r` — Window function exercises
- `02_rank_care_events.sql` / `.r` — Ranking care events
- `03_ctes.sql` / `.r` — Common Table Expressions (CTEs)
- `R_to_SQL.QMD` — Quarto document comparing R and SQL approaches
- `generate_lcs_data.py` — Script to generate LCS sample data
- `load_lcs.data.sql` — SQL script to load LCS data into DuckDB
- `lcs.duckdb` / `lcs_data.duckdb` — DuckDB database files
